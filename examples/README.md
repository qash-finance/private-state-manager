# Examples

Small projects that showcase different ways to interact with Private State Manager:

- [`demo/`](./demo/) – Terminal UI for hands-on experimentation (recommended starting point).
- [`rust/`](./rust/) – Low-level Rust example with binaries for both local-node and mockchain flows.

All examples expect a running PSM server (`cargo run -p private-state-manager-server --bin server`). Some also require a Miden node on `http://localhost:57291`. See each subdirectory README for specific instructions.
